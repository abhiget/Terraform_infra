# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-28

### Added
- Initial Hub-Spoke network architecture
- VNet module with configurable subnets
- Azure Firewall module
- Network Security Groups module
- User Defined Routes module
- Azure Bastion host module
- Monitoring module with Log Analytics integration
- Deployment scripts for automation
- Comprehensive documentation
